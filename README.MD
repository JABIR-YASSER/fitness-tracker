# Fitness Tracker 💪

Full-stack web application built with:

## 🧰 Tech Stack
- Frontend: React + Tailwind CSS
- Backend: PHP (Laragon)
- Database: MySQL

## 🚀 Features
- Add / Edit / Delete workouts
- Charts & analytics
- Search & filters
- Dark mode
- Authentication

## 📁 Structure
frontend/ → React app  
backend/ → PHP API  

## ▶️ Run locally

### Frontend
cd frontend  
npm install  
npm run dev  

### Backend
Place backend folder inside Laragon www  
Start Laragon
